1. Introduction

Web browsers come with built-in features to handle forms without requiring JavaScript. These features include:

  • Automatic form submission with action and method
  • Built-in validation using required, type, pattern
  • Default input controls like text, email, password, number
  • Accessibility and mobile-friendly input types

2. Basic Form Structure

  <label for="email">Email:</label>

4. Autofill, Autocomplete & Placeholders

Browsers can autofill inputs when using autocomplete.

<input type="text" name="username" autocomplete="username" placeholder="Enter your username">
<input type="password" name="password" autocomplete="current-password" placeholder="Enter password">
