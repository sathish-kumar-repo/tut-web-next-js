Convert React.js Project to Next.js Project

A Complete Step-by-Step Tutorial

Introduction

React.js is a powerful library for building UI components, but when it comes to SEO, routing, and server-side rendering, Next.js provides a more complete framework. In this tutorial, we‚Äôll go step by step on how to convert an existing React.js project into a Next.js project.

Step 1: Create a New Next.js Project

First, install Next.js using the official CLI:

npx create-next-app@latest my-next-app

Navigate to the new project folder:

cd my-next-app

Step 3: Move Components

Copy everything inside your React src/components folder into the components folder in your Next.js app. 


cp -r ../my-react-app/src/components ./components

Step 4: Move Assets

If you have images, icons, or static files in public/, copy them into the public/ folder in your Next.js app.

Step 5: Setup Pages

In React, you typically use react-router-dom for routing. In Next.js, each file inside pages/ becomes a route.

Example:


// pages/index.js
import React from "react";
import Header from "../components/Header";

export default function Home() {
  return (
    <div>
      <Header />
      <h1>Welcome to Next.js!</h1>
    </div>
  );
}

Step 6: Remove React Router

Since Next.js has built-in routing, you don‚Äôt need react-router-dom. Uninstall it: 

npm uninstall react-router-dom

Use Next.js Link instead:


import Link from "next/link";

export default function Navbar() {
  return (
    <nav>
      <Link href="/">Home</Link>
      <Link href="/about">About</Link>
    </nav>
  );
}

Step 7: Convert index.js

In React, you have an index.js rendering the <App />. In Next.js, you don‚Äôt need it.

Instead, each page in pages/ is automatically rendered.

Step 8: Global Styles

Move your global CSS (like App.css) into styles/globals.css and import it inside _app.js. 


// pages/_app.js
import "../styles/globals.css";

export default function MyApp({ Component, pageProps }) {
  return <Component {...pageProps} />;
}

Step 9: Environment Variables

Rename your .env variables:

  • React: REACT_APP_API_URL
  • Next.js: NEXT_PUBLIC_API_URL

Step 10: Run and Test

Start your Next.js project:

npm run dev

Open http://localhost:3000 in your browser.
